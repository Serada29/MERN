Nice setup 👍 — mastering shortcuts will **10x your speed** in VS Code.

Here are the **most useful CTRL shortcuts** (focused on what you’ll actually use daily 👇)

---

# ⚡ ESSENTIAL NAVIGATION

### 🧠 Move fast between files & tabs

- `Ctrl + Tab` → Switch between recent tabs
- `Ctrl + Shift + Tab` → Reverse switch
- `Ctrl + 1 / 2 / 3` → Focus editor groups (split views)
- `Ctrl + W` → Close current tab
- `Ctrl + Shift + T` → Reopen closed tab

👉 **Pro tip:** Think of `Ctrl + Tab` like ALT+TAB for code

---

# 🔍 SEARCH LIKE A PRO

- `Ctrl + P` → Quick file search (VERY IMPORTANT 🔥)

  > Type file name → instant open

- `Ctrl + Shift + F` → Search in whole project
- `Ctrl + F` → Search in current file
- `Ctrl + H` → Replace

👉 This alone saves hours vs clicking folders

---

# ✍️ EDITING SPEED

### Multi-cursor & fast edits

- `Alt + Click` → Add multiple cursors
- `Ctrl + D` → Select next same word
- `Ctrl + Shift + L` → Select all same words
- `Ctrl + X` → Cut line (no selection needed)
- `Alt + ↑ / ↓` → Move line up/down
- `Shift + Alt + ↓` → Duplicate line

---

# ⚙️ CODE ACTIONS

- `Ctrl + /` → Comment / uncomment
- `Ctrl + Space` → Trigger suggestions
- `Ctrl + Shift + Space` → Parameter hints
- `F2` → Rename variable everywhere
- `Ctrl + .` → Quick fix (VERY useful 🔥)

---

# 📂 FILE MANAGEMENT

- `Ctrl + N` → New file
- `Ctrl + S` → Save
- `Ctrl + Shift + S` → Save as
- `Ctrl + B` → Toggle sidebar (Explorer)

---

# 🧭 TERMINAL & RUNNING

- `Ctrl + `` → Open terminal
- `Ctrl + Shift + `` → New terminal
- `Ctrl + C` → Stop running script

---

# 🔥 ADVANCED (HIGH VALUE)

- `Ctrl + Shift + P` → Command Palette

  > You can do almost anything here

- `Ctrl + G` → Go to line
- `F12` → Go to definition
- `Alt + F12` → Peek definition
- `Ctrl + Shift + O` → Go to symbol (functions, variables)

---

# 🧠 REAL WORKFLOW (IMPORTANT)

Instead of:

> clicking folders → clicking files

Do this:

1. `Ctrl + P`
2. type `variables`
3. press Enter

⚡ Done in 1 second
